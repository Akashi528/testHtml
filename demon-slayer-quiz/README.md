# 鬼灭之刃 — 柱 测评（前端静态）

这是一个纯前端的测评 Demo：测测你在《鬼灭之刃》里最接近哪位柱（9 位柱 + 头柱炭治郎，共 10 位）。

项目结构

- index.html — 主页面与 SPA 路由
- styles.css — 简约风样式
- app.js — 核心逻辑、题库与评分
- assets/images/ — 占位图片（pillar_1.svg ~ pillar_10.svg），请替换为你自己的图片

使用说明

1. 打开 `index.html`（直接在浏览器中打开文件即可）。
2. 在主页选择“简易版（20 题）”或“完整版（60 题）”。
3. 每题一页（通过浏览器哈希路由实现），完成后将显示与你最匹配的柱与描述，并显示图片占位。

替换占位图片

占位图片位于 `assets/images/pillar_1.svg` 到 `pillar_10.svg`。建议替换为你准备的 JPEG/PNG 或 SVG：

- 保持文件名不变：`pillar_1.*` 对应第 1 位柱（索引 0），以此类推，`pillar_10.*` 对应第 10 位（炭治郎）。
- 支持格式：.svg、.png、.jpg
- 建议尺寸：至少 400×400px，以保证显示质量。

如果你要用自己的图片，直接覆盖这些文件或改名并在 `app.js` 中调整 `resultImg.src` 的生成规则。

自定义/扩展

- 想要每个柱更丰富的介绍，可打开 `app.js` 修改 `PILLARS` 数组的 `desc` 字段。
- 想要更复杂的评分（例如权重分数），可在 `fullQuestions` 的每个选项下 `gives` 改为 `{pid:weight}` 的形式并修改 `computeResult`。
