性压抑自评量表（HTML 版）

说明
- 本项目为本地静态页面，可直接在浏览器打开 `index.html`。
- 80 道题，男女通用；页面以“每题一页”的视觉方式呈现（通过 JavaScript 切换）。
- 测验仅供自我参考，非临床诊断工具。如有需要，请咨询专业人员。

使用方法
1. 在文件管理器中双击打开 `index.html`，或在浏览器中使用“文件 > 打开文件”方式打开。
2. 点击选项作答，或者点击“下一题/上题”导航。完成后点击提交查看结果。
3. 可点击“保存结果（JSON）”按钮将原始答题数据导出。

注意
- 本测验题目为敏感但非露骨描述，旨在帮助自我觉察。请在安全与合法的前提下使用。
