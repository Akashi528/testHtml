# 情侣心跳 — 手机触控情侣小游戏


一个纯前端的小型情侣小游戏，适合两人一机或两指同玩，支持手机触控，无需后端。

新增功能（增强版）：
- 背景音乐（轻柔氛围）可开/关。
- 关卡系统：轻松 / 普通 / 挑战，不同节奏与容错度。
- 本地高分榜：将分数保存到 localStorage，显示前 10 名。
- 更多视觉特效：心形光晕、粒子火花与庆祝彩带（confetti）。

玩法简介：
- 屏幕左右两侧分别为两位玩家的触控区域（你 / TA）。
- 点击或触摸时需要交替触碰且保持稳定节奏（以心跳中心为反馈）。
- 在节奏同步内触碰会提升分数，获得更多分数会触发庆祝特效，达到关卡门槛会触发更大的庆祝。

特点：
- 无需安装，仅用 HTML/CSS/JS；开箱即玩。
- 触摸与点击友好，支持手机振动反馈（若设备支持）。
- 视觉友好：渐变、心形动画、彩带特效、粒子与光晕。

如何运行：
1. 在本地打开 `testHtml/couple-game/index.html`（手机端直接用浏览器打开该文件，或把目录放到任意静态服务器上）。
2. 竖屏使用体验最佳。点击“开始”开始游戏，左右交替触摸区域进行游玩。

提示与测试：
- 如果在桌面端测试，可按 `A` 键模拟左侧触碰，按 `L` 键模拟右侧触碰，空格开始。 
- 在游戏界面可以选择关卡并开关音乐。完成游戏后可以在高分榜处输入名字并保存分数（保存到浏览器的 localStorage）。

Enjoy ❤
